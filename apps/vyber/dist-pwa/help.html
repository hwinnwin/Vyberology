VybeR Help

Welcome to VybeR, the AI-powered browser for the Vybe ecosystem.

Getting started

  • Use the address bar to search or enter a URL.
  • Open the AI command palette with Cmd+Shift+K (or Ctrl+Shift+K).
  • Open settings with Cmd+, (or Ctrl+,) to add your AI key.

Install on your device

  • Desktop: use your browser install option when prompted.
  • iOS: tap Share and select Add to Home Screen.

Offline use

VybeR supports offline access after the first load. AI features require a connection.
